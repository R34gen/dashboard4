# Olist Customer Intelligence Dashboard

Streamlit dashboard for Olist Customer Intelligence project.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Required data folder

Keep these files inside `olist_output/`:
- kpi.csv
- monthly_kpi.csv
- category_kpi.csv
- state_kpi.csv
- segment_priority.csv
- rfm_diagnostic.csv
- late_review_impact.csv
- model_comparison.csv
- model_result.csv
- risk_summary.csv
- insight_table.csv
